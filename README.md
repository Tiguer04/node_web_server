# Web Server en NODEJS
Se ha realizado un servidor web que sirve una web similar a twitter.

## ¿Cómo usarlo?
1. Debes hacer el clonado `git clone https://github.com/Tiguer04/node_web_server`.
2. Abrir tu editor de código y escribir en la terminal `npm install`.
3. Definir tu variable de entorno `.env` teniendo en cuenta la plantilla disponible. Recomendación(`PORT=3000` `PUBLIC_PATH=public`).
4. Ejecuta el proyecto con `npm start`.
